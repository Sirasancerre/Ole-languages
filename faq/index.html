¿I would like to start as soon as possible. When is the next start date?

If you already have a basic understanding of Spanish you can start at any given Monday throughout the year. Please make sure to enrol as early as possible since the groups fill up quickly especially during the summer. If you have not studied Spanish before and you would like to participate in a course for absolute beginners, we offer you the possibility to start every other Monday. We have marked the start dates for courses for absolute beginners in orange in the course calendar.
